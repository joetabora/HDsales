# Accessibility Checklist

- [x] Keyboard navigation (Cmd+K command palette)
- [x] Focus visible states on interactive elements
- [x] Semantic HTML in layouts
- [x] Color contrast (dark theme, WCAG AA)
- [x] Form labels on all inputs
- [ ] Screen reader testing
- [ ] ARIA labels on icon-only buttons
- [ ] Skip to main content link
- [ ] Reduced motion preference support
- [ ] Touch target size ≥ 44px on mobile
